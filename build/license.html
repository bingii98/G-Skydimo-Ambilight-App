G Skydimo Ambilight App

Real-time ambilight for Skydimo LED strips

Before you install

  • Requires Windows 10/11 and a Skydimo LED strip connected via USB (CH340).
  • Close the official SkyDimo.exe app before connecting — it locks the COM port.
  • This is an independent open-source project, not affiliated with Skydimo.
